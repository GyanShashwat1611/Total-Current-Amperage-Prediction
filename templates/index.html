POC: Dashboard for time series prediction of Total Current Amperage by the machine categories by power status of the machine



User cases and assumptions

Assumption 1 : The user would like to have the dashboard to predict Total Current Amperage for the future dates so that they can create a plan about power consumption by the machine and draft a finacial documents.
Assumption 2 : To fulfil assumpton 1 the user would like to have the predicted plot of Total Current Amperage for the next month after already stored data.
Assumption 3 : To fulfil assumpton 1 the user would like to generate dynamic plot by predicting Total Current Amperage for the selected date range and power status of machine using time series prediction model.

Other User Cases

There can be numerous user cases can be drafted for the dashboard by predicting Total Current Amperage for diffrent time intervals. The user stories differs from stakeholders to stakeholders based on their function and usage of the plots and KPI reports from the dashboard.
